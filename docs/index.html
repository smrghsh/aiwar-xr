aiwar.cloud Stereoscopic Video Utility and XR Experience Prototype Tool

This utility helps create stereoscopic video for use at Ars Deep Space. Since designing the stereoscopic video experience can be simulated with VR, it also is WebXR enabled.

  1. Started with some custom three.js boilerplate.
  2. Made a Three.js representation of the project
  3. Added staging and planning markers
  4. VR view helps prototype network positioning and shape
  5. Render button exports 4 stereoscopic videos, each at parametrized IPD
  6. CLI FFmpeg based script processes the batch for Ars' Deep Space
